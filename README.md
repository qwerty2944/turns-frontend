# Turns — Web Frontend

보드게임 크로스플레이 플랫폼의 웹 프론트엔드 (Next.js + Colyseus SDK + Phaser).

- 백엔드: https://github.com/qwerty2944/turns-backend (Colyseus, 로컬 작업 폴더 `backend/`)
- 모바일 앱: https://github.com/qwerty2944/turns-app (Flutter, 로컬 작업 폴더 `mobile/`)

```bash
npm install
npm run dev            # http://localhost:3000
./scripts/build-mobile-web.sh   # 앱 웹뷰용 정적 번들(../mobile/assets/web.zip)
```
